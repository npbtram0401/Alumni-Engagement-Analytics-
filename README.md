# University of Greenwich Alumni Engagement Analytics

A data-driven project analyzing donor and alumni engagement using CRM, Facebook, and Twitter data fully conducted in Excel. It covers data cleaning, exploratory analysis, and interactive dashboard development to uncover insights on audience demographics, content performance, and engagement behavior.

Step 1: Data Cleaning & Preparation - Cleaned and structured raw data in Excel (handling missing values, inconsistent formats, and data types) to ensure accuracy and consistency.

Step 2: Exploratory Data Analysis (EDA) - Used Excel functions and charts to analyze patterns in demographics, content performance, and engagement behavior.

Step 3: Dashboard Development – Built interactive dashboards in Excel to visualize insights across CRM and social media platforms.

Step 4: Insights & Recommendations – Interpreted key findings to provide actionable recommendations on content strategy, audience targeting, and optimal posting timing.
